# Papyrus Hello

![Build & Publish](https://github.com/hownee/papyrus-hello/actions/workflows/publish.yaml/badge.svg)

Shared library package for the Papyrus ecosystem. Provides common utilities, types and helpers used across Papyrus micro-frontends and extensions.

## Installation

```bash
pnpm add @hownee/papyrus-hello
```
